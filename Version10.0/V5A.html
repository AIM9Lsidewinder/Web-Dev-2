Learn how to use Bootstrap

In web design, people use bootstrap to help organize their text which is how this text looks nice.

Steps to use Bootstrap:

  1. Import Bootstrap by using <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.2/css/bootstrap.css"> to import all the things we need
  2. Style the Web page
  3. Use Bootstrap classes: Bootstrap provides utility classes that can be used to style the footer quickly. For example, classes like .bg-dark, .text-white, and .text-center can be used to style the footer.
